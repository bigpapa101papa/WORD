Overview

Computer hardware refers to the physical components of a computer system that enable it to perform various tasks. These components include the CPU, RAM, storage devices, motherboard, GPU, PSU, input and output devices, cooling systems, and expansion cards.

Key Concepts

  • CPU (Central Processing Unit): The CPU is the central component of a computer that executes instructions and processes data.
  • RAM (Random Access Memory): RAM temporarily stores data and instructions that the CPU needs to access quickly.
  • Storage Devices: HDDs and SSDs are common types of storage devices used to permanently store data.
  • Motherboard: The motherboard connects and provides power to various components such as the CPU, memory, and expansion cards.
  • GPU (Graphics Processing Unit): The GPU is responsible for rendering images and videos on the computer screen.
  • PSU (Power Supply Unit): The PSU converts electricity from the wall outlet into a form that can be used by the computer's components.
  • Input and Output Devices: These devices allow users to interact with the computer and display results of processing.
  • Cooling Systems: Fans, heat sinks, and liquid cooling solutions prevent components from overheating.
  • Expansion Cards: Additional circuit boards that add functionality to the computer, such as network interface cards or sound cards.
